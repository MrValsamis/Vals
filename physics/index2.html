About

This site contains course notes for algebra based physics with explorable explanations. I'm still adding content for 2022-2023. Let me know if you have any ideas or if you found one of the numerous mistakes.

avalsamis@glenbrook225.org


The diagrams are SVG written in a text editor.
The math is rendered with the typesetting library KaTeX.
The simulations are JavaScript outputting to canvas or SVG.
Collision simulations use the Matter.js 2-D physics engine.
